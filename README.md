# ChipsSEMDensoising


## Installation

To set up the environment, run:

### Using Conda
```bash
conda env create -f environment.yml
conda activate semdenoising
```
### Using source

```bash
python -m venv chipSEMdenoising
source chipSEMdenoising/bin/activate
pip install -e .
```
